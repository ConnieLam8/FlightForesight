# FlightForeSight: Machine Learning Prediction Model

Airplane delays compromise a significant number of flights annually, negatively impacting passengers. We aim to develop a machine learning model capable of accurately predicting flight delays. This predictive tool will be made accessible to customers, helping them make more informed decisions when planning their travel.


## Features

- Search For Flights
- Predict Delays

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`FLIGHT_API_KEY`
`MYSQL_HOST`
`MYSQL_USER`
`MYSQL_PASSWORD`
`MYSQL_DATABAS`
`SERVER_ML_URL`
